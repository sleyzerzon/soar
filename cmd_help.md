# help #

Provide formatted usage information about Soar commands.

## Synopsis ##

```
help [command_name]
```

### Default Aliases ###

| `?` | `help` |
|:----|:-------|
| `h` | `help` |
| `man` | `help` |

## Description ##

This command prints formatted help for the given command name. Issue alone to
see what topics have help available.
