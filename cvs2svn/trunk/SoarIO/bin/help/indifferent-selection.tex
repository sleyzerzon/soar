\subsubsection*{From Soar Wiki}

\section*{ Name }


 \textbf{indifferent-selection}
 - Controls indifferent preference arbitration. 


 Priority: 4 ; Status: Complete\\ 
--ask gets set but the actual asking does not work at all, I think it locks everything up--Jonathan 13:29, 23 Feb 2005 (EST) 
\section*{ Synopsis }
\begin{verbatim}
indifferent-selection [-aflr]

\end{verbatim}
\section*{ Options }


\begin{tabular}{|c|c|}
\hline 
 -a, --ask  & Ask the user to choose.  \\
 \hline 
 -f, --first  & Select the first indifferent object from Soar's internal list.  \\
 \hline 
 -l, --last  & Select the last indifferent object from Soar's internal list.  \\
 \hline 
 -r, --random  & Select randomly (default).  \\
 \hline 

\end{tabular}



 \\ 

\section*{ Description }


 The indifferent-selection command allows the user to set which option should be used to select between operator proposals that are mutally indifferent in preference memory.
