Popd

Name

popd - Pop the current working directory off the stack and change to the next directory on the stack. Can be relative pathname or fully specified path.

Status: Complete

Synopsis 

popd

Options

No options.

Description

Soar8.3 Description

This command pops a directory off of the directory stack and cd's to it. See the dirs command for an explanation of the directory stack.

Examples


See Also

cd dirs home ls pushd popd source topd
