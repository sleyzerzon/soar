Name

help - Provide formatted usage information about Soar commands.

Status: Incomplete
Currently uses working directory to find command-names and help/ subdir.--Jonathan 14:02, 25 Mar 2005 (EST)

Synopsis 

help [command_name]

Options

command_name Print usage syntax for the command.


Description

This command prints formatted help for the given command name.

Examples

To see the syntax for the excise command: 

help excise

To see what commands help is available for: 

help
