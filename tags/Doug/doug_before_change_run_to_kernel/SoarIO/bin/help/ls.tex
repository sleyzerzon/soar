\section*{ Name }


 \textbf{ls}
 - List the contents of the current working directory. 


 Status: Complete\\ 
I'm saying complete here but the ls command does not support search refinement, etc. (e.g. ls *.cpp)--Jonathan 16:01, 3 Feb 2005 (EST)
